# dlwf-self-supervised
Self-supervised learning adaptation for the DLWF website fingerprinting dataset

## Goal
Adapt the following self-supervised learning methods to the DLWF dataset:
- TS2Vec
- SimCLR
- CPC
- MAE
- BYOL

## Repository Structure
- `data/`: dataset and preprocessing scripts
- `models/`: model implementations and wrappers
- `configs/`: experiment and training configurations
- `notebooks/`: exploration notebooks
- `progress/`: weekly research progress updates
- `README.md`: project overview

## Weekly Updates
Progress updates will be tracked in markdown files in this repository.
